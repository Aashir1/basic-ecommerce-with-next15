# Getting Started

## Prerequisites
Make sure you have **Node.js v18.18.0** or above installed on your system.

## Installation

### Step 1: Install Dependencies
For **npm** users:
```sh
npm install
```
For **yarn** users:
```sh
yarn add
```

### Step 2: Start the Development Server
For **npm** users:
```sh
npm run dev
```
For **yarn** users:
```sh
yarn dev
```
